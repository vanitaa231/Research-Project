Download the dataset for project:
1. Download the Retail_Transaction_Dataset from Kaggle " https://www.kaggle.com/datasets/prasad22/retail-transactions-dataset "
2. Add the dataset in data folder

Run the project:
 python app.py

 Machine Learning Algorithms used:
 1. Holt-Winter
 2. Linear Regression
 3. Decision Tree

Performance Check Matrics
1. MAE
2. RMSE
3. MAPE
